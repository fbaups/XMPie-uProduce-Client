# XMPie uProduce Client Library

XMPie uProduce client to perform hig level tasks against the API.

Provides a wrapper to perform common actions such as:

- Create a new Account/Camping
- Create a new Datasource
- Upload a new Document
- Execute a production/proof run

The goal is to provide high level functions for some common tasks - but not everything...
